Standing on the Shoulders of Giants¶

r0astr is a multi-panel live coding interface built on top of these projects:

  • ꩜ Strudel by Felix Roos

    Tidal Cycles ported to JavaScript — the engine under r0astr's hood.

    ꩜ Strudel Learn ꩜ Pattern Reference

  • Tidal Cycles by Alex McLean

    The pattern language that started it all.

    Tidal Cycles Docs

For pattern syntax, mini notation, samples, scales, and effects, go straight to the source.

Learning Path¶

  1. Start with Concepts — Understand how r0astr organizes instruments
  2. Learn Strudel patterns — Work through the Strudel workshop
  3. Explore Multi-Instrument — Combine panels for full arrangements
  4. Use the Master Panel — Control global parameters
  5. Try an arrangement — Copy a full track from Example Arrangements
