[0.8.1] - 2025-12-14¶

Added¶

  • Complete documentation for Epics 3-5 (12 stories)
  • Remote control comprehensive guide with setup instructions
  • Plugin development guide with Hello World example
  • Showcase page with featured creations and pattern examples
  • Skins system CSS variables reference
  • FAQ page with 15+ entries across 5 categories

Changed¶

  • Enhanced extend/skins.md with theme examples
  • Improved community/showcase.md with tabbed pattern examples

[0.8.0] - 2025-12-13¶

Added¶

  • Major UI update with improved panel styling
  • Focused panel highlighting
  • Better visual feedback for playing/stopped states
  • Comprehensive documentation site

Changed¶

  • Refined panel layout and spacing
  • Improved button styling and interactions
  • Removed animations for cleaner experience

Fixed¶

  • Icon paths in Electron builder
  • Various UI polish issues

[0.7.0] - 2024-XX-XX¶

Added¶

  • Electron app builds for macOS, Windows, Linux
  • CI/CD pipeline for automatic releases
  • GitHub Pages deployment

Changed¶

  • Build system improvements

[0.6.0] - 2024-XX-XX¶

Added¶

  • Remote control interface (remote.html)
  • WebSocket server for real-time synchronization
  • Stop All functionality

Changed¶

  • Improved state management

[0.5.0] - 2024-XX-XX¶

Added¶

  • Master panel with global controls
  • TEMPO slider
  • Global effect sliders

[0.4.0] - 2024-XX-XX¶

Added¶

  • Slider controls in patterns
  • Real-time parameter adjustment

[0.3.0] - 2024-XX-XX¶

Added¶

  • Multi-panel interface
  • Independent panel control
  • Synchronized audio clock

[0.2.0] - 2024-XX-XX¶

Added¶

  • Basic Strudel integration
  • Pattern evaluation
  • Web Audio output

[0.1.0] - 2024-XX-XX¶

Added¶

  • Initial project setup
  • Basic UI structure
