# Story 2.4: Settings Modal and Secondary UI

## Status
Done

## Story
**As a** user,
**I want** the settings modal, menus, metronome, banner bar, and code editor theme to match the glassmorphism design with proper layering and blur effects,
**so that** the entire interface feels unified from primary panels to secondary chrome.

## Acceptance Criteria
1. **Settings modal:** frosted glass panel (from Story 2.3 modal pattern), tab styling with green active indicator (`border-bottom: 2px solid var(--color-primary)`), inactive tabs use `var(--text-muted)`
2. **Settings controls:** toggles use `var(--color-primary)` when active, `var(--surface-card-80)` when inactive; dropdowns and inputs follow Story 2.3 input pattern
3. **Metronome section:** step boxes use `var(--surface-card-50)` background, active step uses `var(--color-primary)` with subtle green glow, beat indicator text in `var(--text-secondary)`
4. **Banner bar:** translucent background `var(--surface-card-30)`, logo and subtitle use `var(--text-secondary)`, no solid background color
5. **Top menu bar:** `backdrop-filter: blur(12px)` when sticky, `var(--surface-card-50)` background, buttons styled as ghost buttons (Story 2.3 pattern)
6. **CodeMirror editor theme:** dark background matching `var(--color-background)`, cursor color `var(--color-primary)`, selection highlight `oklch(0.55 0.15 150 / 0.2)`, gutter background `var(--surface-card-30)`, line numbers `var(--text-muted)`
7. **Add-panel button:** dashed border using `var(--border-emphasis)` (`oklch(1 0 0 / 0.2)`), text in `var(--text-muted)`, hover brightens border and text opacity
8. **Toast notifications:** `var(--surface-card-80)` background with `backdrop-filter: blur(4px)`, `var(--border-default)` border, `border-left: 4px solid {status-color}` where status color is semantic (green/amber/red/blue)
9. **Icon sizing:** all Material Icons follow consistent sizing -- 12px inline list items, 14-16px buttons, 18-20px modal headers -- with `display: inline-flex; align-items: center; gap: var(--gap-sm)` pattern
10. All colors in oklch -- no hex/hsl

## Dev Notes

### CodeMirror Theme Integration
The CodeMirror editor is initialized in `src/panels/panelEditor.js`. The editor theme can be customized via:
1. A CodeMirror theme extension (JavaScript, applied during editor creation)
2. CSS overrides targeting `.cm-editor`, `.cm-gutters`, `.cm-activeLineGutter`, `.cm-cursor`, `.cm-selectionBackground` etc.

Option 2 (CSS overrides) is preferred for this story since it keeps styling in the skin CSS layer. Use `!important` sparingly -- prefer high-specificity selectors.

### Key Files
- `static/css/components.css` (settings modal, toggles, toasts)
- `static/css/base.css` (banner bar, menu bar, metronome)
- `public/skins/default/theme.css` (skin-specific overrides)
- `index.html` (settings modal HTML structure, metronome structure)
- `src/panels/panelEditor.js` (CodeMirror initialization -- reference only)

### Settings Modal Structure (from index.html)
The settings modal has these tabs:
- General (skin selector, layout options)
- Editor (font size, line numbers, word wrap)
- Behaviour (auto-play, auto-save)
- Integrations (WebSocket, OSC)
- Accessibility (focus indicators, screen reader)
- Advanced (debug, reset)

Each tab section has various control types: toggles, selects, text inputs, range sliders.

### Technical Constraints
- The metronome step boxes are dynamically created -- styling must target the class, not specific element IDs
- Banner bar height is defined by `--banner-bar-height` (50px) -- preserve this
- Top menu bar auto-hide uses CSS `:hover` on a trigger zone -- preserve this behavior
- CodeMirror generates its own DOM structure -- use `.cm-*` class selectors
- Toast notifications may be dynamically inserted -- ensure styles work for both existing and newly created toasts
- Material Icons font must remain loaded (referenced by `--font-family-icons`)

### Icon Sizing Reference
| Context | Size | Examples |
|---------|------|---------|
| Inline list items | 12px | Status indicators, metadata |
| Buttons | 14-16px | Play, delete, duplicate, settings |
| Modal headers/close | 18-20px | Modal close X, dialog title icons |
