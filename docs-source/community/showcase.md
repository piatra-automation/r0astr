## Submit Your Work

Share your r0astr creations:

### What to Include

- **Title** - Name your creation
- **Description** - Brief explanation of your approach
- **Media** - Video, SoundCloud, or audio file
- **Patterns** - (Optional) Share the patterns you used
- **Your name/handle** - Credit as you'd like to appear

### How to Submit

**Option 1: GitHub PR**

1. Fork the r0astr docs repository
2. Add your entry to this page
3. Include media files or embed links
4. Open a PR

**Option 2: Issue**

1. Open a [GitHub Issue](https://github.com/piatra-automation/r0astr/issues/new)
2. Use title: `[Showcase] Your Creation Title`
3. Include description and links to media

### Guidelines

- Keep it musical - this is about the music you make
- Family-friendly content
- Credit samples if using copyrighted material
- Be supportive of others' submissions
